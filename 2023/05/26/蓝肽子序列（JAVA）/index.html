蓝肽子序列（JAVA）

题目：

L 星球上的生物由蛋蓝质组成，每一种蛋蓝质由一类称为蓝肽的物资首尾连接成一条长链后折叠而成。

生物学家小乔正在研究 L 星球上的蛋蓝质。她拿到两个蛋蓝质的蓝肽序列，想通过这两条蓝肽序列的共同特点来分析两种蛋蓝质的相似性。

具体的，一个蓝肽可以使用 1 至 5 个英文字母表示，其中第一个字母大写，后面的字母小写。一个蛋蓝质的蓝肽序列可以用蓝肽的表示顺序拼接而成。

在一条蓝肽序列中，如果选取其中的一些位置，把这些位置的蓝肽取出，并按照它们在原序列中的位置摆放，则称为这条蓝肽的一个子序列。蓝肽的子序列不一定在原序列中是连续的，中间可能间隔着一些未被取出的蓝肽。

如果第一条蓝肽序列可以取出一个子序列与第二条蓝肽序列中取出的某个子序列相等，则称为一个公共蓝肽子序列。

给定两条蓝肽序列，找出他们最长的那个公共蓝肽子序列的长度。

输入描述
输入两行，每行包含一个字符串，表示一个蓝肽序列。字符串中间没有空格等分隔字符。

其中有 ，两个字符串的长度均不超过 1000。

输出描述
输出一个整数，表示最长的那个公共蓝肽子序列的长度。

输入输出样例
示例

输入

LanQiaoBei
LanTaiXiaoQiao

输出

2

运行限制
最大运行时间：1s
最大运行内存: 128M

思路：

本题是最长公共子序列这一类题目的一个变式。

大家可以先去了解一下最长公共子序列这一类题，这位博主讲的很详细最长公共子序列（点击超链接跳转）

对于本题我们首先要将给定字符串根据大写字母拆分为一个个蓝肽，我将其存储在了LIst集合中，

然后再根据动态规划求最长公共子序列。

解释一下dp[i][j]表示第一个字符串的前i个蓝肽组成的字符串和第二个字符串的前j个蓝肽组成的字符串的最长公共子序列

dp数组初始值dp[0][j]和dp[i][0]这一行一列都为0，因为数组默认值都是0所以就不必再次赋值了。

然后编写动态转移方程

如果最后一个蓝肽相同：

dp[i][j]=dp[i-1][j-1]+1;

如果不同：

dp[i][j]=Math.max(dp[i-1][j-1],dp[i][j-1]);

不理解这两个方程的去看一下上面的最长公共子序列那位博主里面讲的很详细。

最后找最大值并输出就可以了。
