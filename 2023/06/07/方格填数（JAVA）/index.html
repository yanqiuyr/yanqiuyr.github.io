方格填数（JAVA）

题目：

本题为填空题，只需要算出结果后，在代码中使用输出语句将所填结果输出即可。

如下的 10个格子

+–+–+–+
| | | |
+–+–+–+–+
| | | | |
+–+–+–+–+
| | | |
+–+–+–+

填入 0 ~ 9 的数字。要求：连续的两个数字不能相邻。 （左右、上下、对角都算相邻）

一共有多少种可能的填数方案？
